\section*{Supplementary Material}

The following supplementary files accompany this paper.
All numerical data and scripts are available at
\texttt{[repository URL upon acceptance]}.

\subsection*{S1. Code and regression suite}

The complete \texttt{nbody/} package (14 core modules, 166 example
scripts) is provided.  The master reproduction script
\texttt{reproduce\_all.py} executes 10 stages covering all 8 Results
and exits with code~0 in ${\sim}100$\,s on a standard laptop.
See \texttt{MANIFEST\_PUBLICATION.md} for the full file listing.

\subsection*{S2. Output data files}

Selected numerical outputs (CSV format) are provided in
\texttt{examples/\_outputs/}:

\subsection*{S3. Figures (high resolution)}

All figures in the paper are generated by the example scripts and
saved as PNG (150~dpi) in \texttt{examples/\_outputs/}.
Publication-quality versions (300~dpi, PDF) are available upon request.
