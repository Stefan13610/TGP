# Jadra atomowe z solitonow

Dowod czastkowy, ze jadra atomowe (deuteron, tryton, alfa) sa stanami zwiazanymi solitonow Phi. Program numeryczny promowany z `research/` po domknieciu.

## Status
**Domkniety** — 25/25 testow PASS (promowany z `research/` po zakonczeniu programu).

## Zawartosc
- `NUCLEAR_FROM_SOLITON_VERDICT.md` — werdykt domkniecia i podsumowanie dowodu
- `PLAN.md` — plan badawczy programu
- `nfs01_deuteron_two_body.py` — deuteron jako uklad dwucialowy (nfs01)
- `nfs01_fit_params.json` — parametry dopasowania deuteronu
- `nfs02_triton_three_body.py` — tryton jako uklad trojcialowy (nfs02)
- `nfs03_alpha_four_body.py` — czastka alfa jako uklad czterocialowy (nfs03)
- `nfs04_hardcore_verification.py` — weryfikacja rdzenia twardego (nfs04)
- `nfs05_variational_convergence.py` — zbieznosc wariacyjna (nfs05)
