# Dowody czastkowe TGP

Dowody czastkowe rozprawy: dodatki formalizujace pojedyncze twierdzenia, hipotezy lub wyniki numeryczne rozbudowane ponad objetosc sekcji. Zawiera rowniez programy numeryczne promowane z `research/` po domknieciu.

## Podfoldery
- `alphaK_status/` — status epistemiczny parametru alpha_K (Dodatek Z)
- `bh_ringdown/` — ringdown czarnej dziury i ogon WKB (Dodatki C, K)
- `bounce_topology/` — topologia odbicia solitonu od ghost wall (Dodatek T4)
- `chiralnosc/` — chiralna asymetria mas m_R != m_L (Dodatek K2)
- `defect_hierarchy/` — hierarchia defektow topologicznych: dowod formalny (Dodatek D2)
- `fermion_from_soliton/` — fermiony MS jako solitony Phi; program **domkniety 27/27 PASS** (promowany z `research/`)
- `hierarchia_mas/` — hierarchia mas leptonow z wezlow profilu radialnego (Dodatki F, F2, J, J2)
- `koide_fp/` — formula Koidego jako punkt staly fixed-point (Dodatki T, T2, T3, T5, T6)
- `most_gamma_phi/` — most formalny Gamma -> Phi: lematy i update a_Gamma * Phi_0 (Dodatki Q2, W)
- `nuclear_from_soliton/` — jadra atomowe jako stany zwiazane solitonow; program **domkniety 25/25 PASS** (promowany z `research/`)
- `particle_sector/` — sektor czastek: emergencja QM i domkniecie (Dodatki P, P4)
- `potencjal/` — stabilizacja potencjalu i trzecia generacja (Dodatek I_v2)
- `quark_sector/` — sektor kwarkowy: phi-FP a Koide (Dodatek X)
- `superconductivity/` — domkniecie sektora nadprzewodnictwa i poszerzone mechanizmy (Dodatki P5, P6)
- `trojcialowe_nbody/` — nieredukowalne sily trojcialowe i dynamika N-cial (Dodatki D, Y)
- `wielki_wybuch/` — Wielki Wybuch jako przejscie fazowe S_0 -> S_1 (Dodatek G)
- `zero_mode/` — masa solitonu jako korekcja trybu zerowego A4 i dynamiczne uzasadnienie Koidego (Dodatki R, R2)
