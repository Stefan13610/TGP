# Hierarchia mas leptonow

Hierarchia mas leptonow (e, mu, tau) jako wezly profilu radialnego solitonu Phi: dokladny potencjal Sturma-Liouville'a, ogon oscylacyjny i formalizacja sciezki 9 (self-consistent phi-FP).

## Status
Twierdzenie czastkowe / program (D + P).

- `dodatekF_hierarchia_mas.tex` — Dodatek F: hierarchia mas z wezlow profilu radialnego
- `dodatekF_v2_wkb_numerics.tex` — Dodatek F2: dokladny potencjal SL dla fluktuacji kinku, numeryka WKB
- `dodatekJ_ogon_masy.tex` — Dodatek J: amplituda ogona oscylacyjnego a hierarchia mas
- `dodatekJ2_sciezka9_formalizacja.tex` — Dodatek J2: formalizacja sciezki 9 (self-consistent phi-FP)
