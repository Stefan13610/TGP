# Notacja i slownik formalizmu

Slownik notacji i slownik formulacji TGP: definicje symboli, relacje miedzy formulacjami (substratowa, polowa, solitonowa) i ich domeny stosowalnosci.

## Zawartosc
- `dodatekA_notacja.tex` — Dodatek A: slownik notacji (symbole, konwencje, jednostki)
- `slownik_formalizmu.tex` — slownik formalizmu TGP (formulacje i ich domeny)
