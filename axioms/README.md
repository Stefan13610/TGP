# Aksjomaty TGP

Zbior fundamentalnych aksjomatow, na ktorych opiera sie Teoria Generowanej Przestrzeni (TGP): notacja, akt roznicy ustanawiajacy substrat oraz wlasciwosci samego substratu.

## Podfoldery
- `notacja/` — notacja i slownik formalizmu (Dodatek A)
- `roznica_N0/` — aksjomatyka roznicy i nicosci N_0 (Dodatek 0)
- `substrat/` — substrat jako aksjomat fundamentalny (Dodatek B)
