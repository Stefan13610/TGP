## Overview

TGP (Teoria Generowanej Przestrzeni / Topology-Generated Potential) is a scalar-field theory
in which **space is not a fixed background** but is generated by the presence of mass and energy.
Each fundamental particle produces a local contribution to a shared field $\Phi$, whose value
measures the "density of generated space." Nonlinear interference of multiple sources produces
three distinct regimes of interaction:

1. **Long range** -- attraction (gravity)
2. **Intermediate** -- repulsive barrier (high spatial potential)
3. **Short range** -- confining well (binding)

The fundamental constants $c$, $\hbar$, $G$ are field-dependent, with the Planck length held fixed.
Dark energy arises naturally from fluctuations of generated space amplified by matter condensation.

## Quick start: reproduction

```bash
# 1. Install dependencies (pinned versions for deterministic reproduction)
pip install -r requirements.lock
# Or use minimum versions: pip install numpy>=1.26 scipy>=1.12 matplotlib>=3.8

# 2. Run master reproduction (10 stages, ~100s)
python -m nbody.reproduce_all --quick

# 3. Run full regression suite (59 tests, ~2 min)
python -m nbody.examples.verify_all

# Expected: 10/10 PASS, 59/59 PASS
```
