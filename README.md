## Abstract

TGP (*Teoria Generowanej Przestrzeni*) is a conformal scalar-field theory in which spacetime is not a fixed background but is **generated by the presence of mass and energy**. A single field $\Phi = \langle \hat{s}^2 \rangle$ encodes the local density of generated space; the effective metric $g_{ij} = (\Phi/\Phi_0)\,\delta_{ij}$ and all fundamental constants ($c$, $\hbar$, $G$) become field-dependent, with the Planck length held fixed.

From three inputs — the TGP coupling $g_0^e = 0.86941$, the cosmological constant $\Omega_\Lambda = 0.6847$, and the generation number $N = 3$ — the theory derives **40 quantitative predictions**, of which 21 are confirmed by experiment (18 within $2\sigma$, 3 exact). No kill criteria (0/15) are violated.

**Highlights:**
- $\alpha_s(M_Z) = 0.1190$ (PDG: $0.1180 \pm 0.0009$, $1.1\sigma$)
- Koide constant $K = 2/3$ (exact, from algebraic identity)
- Higgs mass $m_H = 125.31$ GeV (PDG: $125.25 \pm 0.17$, $0.3\sigma$)
- Spectral index $n_s = 1 - 2/N_e = 0.967$ (Starobinsky class)
- Tensor-to-scalar ratio $r = 0.0033$ (Mukhanov-Sasaki, within BICEP/Keck)
- Neutrino mass sum $\Sigma m_\nu = 59.6$ meV (normal ordering only; IO excluded)
- Dark energy $w(z=0) = -0.994$, $\chi^2_\mathrm{TGP} \approx \chi^2_{\Lambda\mathrm{CDM}}$
- Kinetic coupling $\alpha = 2$: algebraic theorem, not a fit
- GW prediction: scalar breathing mode (3rd polarization, testable by ET/LISA)

## Quick start

### Requirements

- Python >= 3.10
- LaTeX (pdflatex + bibtex; TeX Live or MiKTeX)
- OS: tested on Windows 11; should work on Linux/macOS

### Reproduce numerical results

```bash
# Install dependencies
pip install -r requirements.lock

# Quick reproduction (10 stages, ~100s)
python -m nbody.reproduce_all --quick

# Full regression suite (59 tests, ~2 min)
python -m nbody.examples.verify_all

# Cross-validation scripts
python scripts/a1_alpha2_frg_synthesis.py    # α=2 theorem (7/7 PASS)
python scripts/c2_neutrino_mass_summary.py   # Neutrino masses (6/6 PASS)
python scripts/b1_wde_friedmann_fit.py       # DESI fit (7/7 PASS)
```

## Falsifiability

TGP is explicitly falsifiable. Any of the following would rule it out:

- **Inverted neutrino mass ordering** confirmed by JUNO/Hyper-K
- **Proton decay** observed at any rate
- **$w_\mathrm{DE} < -1$** (phantom crossing) confirmed at $> 5\sigma$
- **$c_\mathrm{GW} \neq c$** at $> 5\sigma$
- **No breathing mode** in GW ringdown after $> 1000$ events with 3+ detectors
