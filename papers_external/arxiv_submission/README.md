- `ARXIV_METADATA.md` — metadane submission (tytul, abstract, kategorie arXiv)
- `arxiv_submission.tar.gz` — spakowane archiwum do uploadu arXiv
- `tgp_nbody_results_clean.tex` — glowny tekst paperu
- `tgp_nbody_results_clean.pdf` — skompilowany PDF
- `tgp_nbody.bib` — bibliografia
- `supplementary_material.tex` — material suplementarny
- ... oraz pliki pomocnicze LaTeX (aux, bbl, log, out, toc)
