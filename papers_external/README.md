# Papery zewnetrzne TGP

Preprinty i artykuly towarzyszace rozprawie: submission na arXiv oraz osobne papery z podsektorow (czarne dziury, masy leptonow, nadprzewodnictwo, core paper, angielskie streszczenie).

## Podfoldery
- `arxiv_submission/` — submission na arXiv (tgp_nbody_results_clean) z material suplementarnym
- `paper_bh_shadow/` — paper o cieniu czarnej dziury (tgp_bh_shadow)
- `paper_lepton_masses/` — paper o masach leptonow (tgp_lepton_masses)
- `tgp_core_paper/` — core paper TGP (tgp_core)
- `tgp_english_summary/` — angielskie streszczenie rozprawy (main_en, sec01-sec09)
- `tgp_sc_paper/` — paper o nadprzewodnictwie (tgp_sc)
