# Dodatki formalizmu TGP

Dodatki formalizujace rdzen teorii: kwantyzacja pola Phi, grupy homotopii solitonu, lancuch wyprowadzen A1->K20, stabilizacja i renormalizacja ERG, emergencja grup cechowania U(1), SU(2), SU(3) oraz coarse-graining Gamma->Phi.

## Zawartosc
- `dodatekE_kwantyzacja.tex` — Dodatek E: kwantyzacja pola Phi (schemat samozwrotny)
- `dodatekE_pi1_formal.tex` — formalny dowod pi_1(C_sol) = Z_2
- `dodatekH_lancuch_wyprowadzen.tex` — Dodatek H: kanoniczny lancuch wyprowadzen A1->K20
- `dodatekL_formula_sumy.tex` — Dodatek L: formula sumy, skan kinetyczny i stosunek mas leptonow
- `dodatekM_erg_stabilizacja.tex` — Dodatek M: stabilizacja prozni TGP (poza LPA)
- `dodatekN_erg_renormalizacja.tex` — Dodatek N: przeplyw ERG dla substratu TGP
- `dodatekO_u1_formalizacja.tex` — Dodatek O: formalizacja emergencji U(1)
- `dodatekQ_coarse_graining_formal.tex` — Dodatek Q: formalne gruboziarnienie Gamma->Phi
- `dodatekU_su2_formalizacja.tex` — Dodatek U: sektor SU(2) x U(1), masy bozonow
- `dodatekV_su3_formalizacja.tex` — Dodatek V: sektor SU(3)_c, emergencja gluonow
