# Meta-pliki LaTeX rdzenia

Pliki LaTeX meta-poziomu: mapa statusu twierdzen TGP, kanoniczna klasyfikacja epistemiczna oraz nota uzgadniajaca rozne definicje predkosci swiatla.

## Zawartosc
- `status_map.tex` — mapa statusu twierdzen TGP (co jest dowiedzione, co hipoteza, co program)
- `tabela_epistemiczna.tex` — kanoniczna klasyfikacja epistemiczna (K, P, W, D, I, R)
- `nota_c_phi_reconciliation.tex` — nota wyjasniajaca trzy predkosci swiatla w TGP (c_proper, c_lok, c_koord)
