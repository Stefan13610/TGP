# Rdzen teorii TGP

Rdzen Teorii Generowanej Przestrzeni: sekcje 0-10 rozprawy oraz dodatki formalizujace (E, H, L, M, N, O, Q, U, V). Zawiera rowniez meta-pliki LaTeX (mapa statusu, tabela epistemiczna).

## Podfoldery
- `_meta_latex/` — meta-pliki LaTeX (mapa statusu, tabela epistemiczna, nota o predkosciach swiatla)
- `formalizm/` — dodatki formalizujace rdzen (kwantyzacja, pi_1, lancuch wyprowadzen, ERG, U(1), SU(2), SU(3), coarse-graining)
- `sek00_summary/` — Sekcja 0: streszczenie wykonawcze
- `sek01_ontologia/` — Sekcja 1: ontologia (N_0, materia, przestrzen)
- `sek02_pole/` — Sekcja 2: pole Phi i rownanie pola
- `sek03_rezimy/` — Sekcja 3: trzy rezimy oddzialywania
- `sek04_stale/` — Sekcja 4: dynamiczne stale fundamentalne (c, hbar, G)
- `sek05_ciemna_energia/` — Sekcja 5: ciemna energia z fluktuacji Phi
- `sek06_czarne_dziury/` — Sekcja 6: czarne dziury jako lokalna nicosc
- `sek07_predykcje/` — Sekcja 7: predykcje, falsyfikowalnosc, problemy otwarte
- `sek07a_wymiar_wzmocniony/` — Sekcja 7a: wzmocniony argument za d=3
- `sek08_formalizm/` — Sekcja 8: formalizm matematyczny
- `sek08a_akcja_zunifikowana/` — Sekcja 8a: zunifikowana akcja i wyprowadzenie kappa
- `sek08b_ghost_resolution/` — Sekcja 8b: rozwiazanie problemu duchow kinetycznych
- `sek08c_metryka_z_substratu/` — Sekcja 8c: metryka efektywna z budzetu substratu
- `sek09_cechowanie/` — Sekcja 9: emergencja pol cechowania
- `sek10_N0_wyprowadzenie/` — Sekcja 10: formalne wyprowadzenia warunkow N_0
