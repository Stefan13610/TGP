PART A: ANALYTICAL LINEARIZATION OF THE TGP SUBSTRATE FIELD EQUATION
==============================================================================

1. FULL NONLINEAR TGP FIELD EQUATION (spherical symmetry, d=3):
   g''(r) + (2/r) g'(r) + g(r) [1 - g(r)^2] = 0
   Boundary conditions: g(0) = g0 (finite), g'(0) = 0, g(inf) = 1

2. EXPANSION AROUND VACUUM g = 1:
   Let g(r) = 1 + u(r),  where |u| << 1
   Substituting:
   u'' + (2/r) u' + (1+u)[1 - (1+u)^2] = 0
   Expand: (1+u)(1-(1+u)^2) = (1+u)(-2u-u^2) = -2u - 3u^2 - u^3
   Full equation: u'' + (2/r) u' - 2u - 3u^2 - u^3 = 0

3. LINEARIZED EQUATION (drop O(u^2) terms):
   u'' + (2/r) u' - 2u = 0
   This is the MODIFIED HELMHOLTZ equation in spherical coords
   with mass parameter mu^2 = 2.

   NONLINEAR RESIDUAL: R_NL = -3u^2 - u^3 = O(u^2)
   For |u| < epsilon: |R_NL| <= 3*epsilon^2 + epsilon^3

4. GENERAL SOLUTION of the linearized equation:
   u(r) = A * exp(-sqrt(2)*r) / r   (decaying)
        + B * exp(+sqrt(2)*r) / r   (growing, REJECTED)

   Proof: substitute u = f(r)/r:
   u'' + 2u'/r = f''/r  =>  f'' - 2f = 0
   => f = C1*exp(-sqrt(2)*r) + C2*exp(+sqrt(2)*r)
   Physical (bounded at inf): C2 = 0

5. PHYSICAL TAIL SOLUTION:
   u(r) = A * exp(-sqrt(2)*r) / r
   Decay length: L = 1/sqrt(2) ~ 0.707 natural units

   VERIFICATION: max |u''+2u'/r-2u| = 4.44e-16 (machine eps)
