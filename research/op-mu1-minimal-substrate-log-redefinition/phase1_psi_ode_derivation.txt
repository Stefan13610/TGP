  α=2 specific analysis

           = e^(-3ψ) - e^(-2ψ)

RHS w ψ=0: 0  (oczekiwane 0 — vacuum fixed point)

Linearyzacja wokół ψ=0: RHS ≈ -psi
  (ψ ≈ 0, RHS ≈ -ψ — restoring force, vacuum stabilny)

Asymptotyka dla α=2:
  ψ → +∞ (g → ∞):  RHS = e^(-3ψ) - e^(-2ψ) → 0⁻ (oba człony → 0)
  ψ → -∞ (g → 0):  RHS = e^(-3ψ) - e^(-2ψ) → +∞ (singular)

UWAGA: ψ → -∞ odpowiada g → 0 (zerowanie metryki).
Fizyczne soliton solutions z g₀=1.77 → ψ₀=0.57 → ψ ∈ [0, ψ₀] — bezpieczne.
