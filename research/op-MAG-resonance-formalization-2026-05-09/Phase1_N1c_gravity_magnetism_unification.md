## Część II: Mathematical structure proposed

### Phase as connecting concept

W standard wave physics, phase φ(x, t) rozróżnia:
- **In-phase** (Δφ = 0): constructive interference, scalar response
- **Out-of-phase** (Δφ = π): destructive
- **Quadrature** (Δφ = π/2): vector / rotational response

Może to być klucz dla unification:
- Static / co-moving sources: in-phase coupling → scalar gravity-like
- Counter-moving / orthogonal motion: phase difference → vector magnetism-like

```

```
