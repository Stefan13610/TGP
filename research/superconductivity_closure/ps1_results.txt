==============================================================================
  Part F.  Topologiczna ochrona prądu persystentnego
==============================================================================

  Parametr porzadku:  psi(r) = |psi|(r) exp(i theta(r))  (U(1) field)
  Pierwsza grupa homotopii:  pi_1(U(1)) = Z
  => kwantyzowane winding numbers n in Z:  loop_integral(grad_theta . dl) = 2 pi n

  Dla d=3 rozwiciem windingu jest VORTEX-RING o promieniu R.
  Energia ringa (standardowy wynik XY/London):
     E_ring(R) = 2 pi J R [ ln(R/xi_core) - 1 ]

  Bariera rozwicia rosnie liniowo z promieniem R => ekstensywnie z objętoscia.
  Stan n != 0 nie moze rozpasc sie bez pokonania bariery.

  Przyklad: dla J* = 7.5068e-01, xi_core = 1.0
    R =     10:  E_ring = +6.144e+01   E/k_B T_c^SC = 3.72e+01
    R =    100:  E_ring = +1.700e+03   E/k_B T_c^SC = 1.03e+03
    R =   1000:  E_ring = +2.786e+04   E/k_B T_c^SC = 1.69e+04

  Wniosek: bariera E_ring / k_B T_c rosnie z R (roznica skali).
  Makroskopowy prad persystentny (R ~ rozmiar probki) jest topologicznie
  chroniony: E_barrier >> k_B T dla T < T_c.

  (iii) pi_1(U(1)) = Z ? TAK  (struktura U(1) substratu TGP)
